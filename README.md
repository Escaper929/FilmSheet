# FilmSheet

胶片扫描排版工具 —— 把你的数码扫描件变成真实的灯箱灯板 / 接触印相作品。

支持 135 和 120 两种画幅，物理级齿孔还原、自定义包装图、双语标签，一键生成专业级胶片展示图。

## 功能特点

### 🎞️ 双画幅支持

| 画幅 | 支持格式 |
|------|---------|
| **135** | 标准 36×24、半格 18×24、方形 24×24、XPan 65×24 |
| **120** | 6×4.5、6×6、6×7、6×8、6×9、6×12、6×17 |

### 🖼️ 真实还原

- **物理级齿孔** — KS（民用）和 BH（电影）两种齿孔样式，严格遵循 SMPTE 标准尺寸
- **自动齿孔类型识别** — 根据胶卷名称自动判断 KS / BH
- **4x 抗锯齿渲染** — 放大 4 倍绘制后 LANCZOS 缩小，文字和图形边缘丝滑
- **胶片边字** — 模拟真实胶片边栏文字，支持自定义内容
- **随机边字排列** — 每次生成边字位置略有不同，更贴近真实效果

### 📦 包装图叠加

- 支持导入胶卷包装图片（Kodak / Fujifilm 等）
- 左右位置可选，大小滑块调节
- 描边开关，自动适配
- 历史记录最近 30 张

### ✏️ 拍摄信息记录

- 卷号、相机、胶卷、拍摄日期、冲洗日期
- 冲洗方式、冲洗地点、扫描仪
- 中英双语标签一键切换
- 自动排版，超长文字自动缩放

### 🎨 两种渲染风格

| 风格 | 描述 |
|------|------|
| **灯板正片** | 白色背景 + 暖色边字，模拟灯箱观看效果 |
| **接触印相** | 黑色背景 + 白色边字，经典接触印相风格 |

### ⚡ 高效批量

- **多线程并行处理** — 根据 CPU 核心数自动分配线程
- **正片 / 负片模式** — 一键反转色调
- **强制横向** — 竖图自动旋转 90°
- **自适应列数** — 根据画幅自动推荐最佳行列比

## 使用说明

### 1. 准备图片

将你的胶片扫描件放入一个文件夹中，支持 JPG、PNG、TIFF、BMP 格式。建议扫描分辨率 300-600 DPI。

### 2. 基本设置

| 步骤 | 操作 |
|------|------|
| ① | 点击「浏览...」选择图片文件夹 |
| ② | 输入输出文件名（默认 `filmsheet_output.jpg`） |

### 3. 参数设置

- **成像模式** — 正片（原始色调）/ 负片（反转色调）
- **画幅** — 135 或 120，切换后子画幅选项自动更新
- **缩略图宽** — 300-800px，越大越清晰，生成时间越长
- **每行列数** — 手动调整或点击「自适应画幅」自动设置
- **齿孔模式** — Auto（自动识别）/ KS（民用）/ BH（电影）
- **渲染风格** — 灯板正片 / 接触印相

### 4. 边字设置

- 留空则自动生成：`胶卷品牌 型号 ◀`
- 填写自定义内容则使用你输入的文本

### 5. 胶卷包装图（可选）

- 点击「浏览」选择胶卷包装盒照片
- 选择位置：左侧 / 右侧
- 大小滑块控制包装图高度占比
- 勾选「描边」添加边框

### 6. 拍摄信息（可选）

填写你的拍摄和冲洗记录，支持中英文标签切换。未填写的字段会自动隐藏。

### 7. 输出选项

- **格式** — PNG（无损）/ JPG（可调质量）
- **质量** — 1-100，仅 JPG 格式可用

### 8. 开始生成

点击「开始生成」按钮，等待进度条完成。生成结束后会自动打开输出文件夹。

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + Enter` | 开始生成 |
| `Esc` | 取消生成 |
| `Ctrl + O` | 浏览输入文件夹 |

## 常见问题

**Q: 生成的图片模糊怎么办？**

A: 尝试增大「缩略图宽」参数（建议 600-800px），数值越大最终图片越清晰。

**Q: 齿孔不对齐？**

A: 齿孔模式设为 Auto 时会根据胶卷名称自动判断 KS/BH。如果识别不准确，可以手动指定。

**Q: 包装图显示不全？**

A: 确保选择的图片是清晰的胶卷包装照片，宽高比不要太极端。

**Q: 生成的文件名太长？**

A: 输出文件名会自动使用你填写的信息（卷号_相机_胶卷_日期），填写越简洁文件名越短。

## 技术细节

- 渲染引擎基于 SMPTE 307M 标准，齿孔尺寸精确到 0.001mm
- 135 模式支持 4 种子画幅，每种有不同的齿孔数和胶片前进距离
- 120 模式支持 7 种常见画幅，自动计算裁剪比例
- 所有渲染均使用 4x 超采样抗锯齿，输出 LANCZOS 降采样
- 配置文件自动保存在系统标准目录：
  - macOS: `~/Library/Application Support/FilmSheet/`
  - Windows: `%APPDATA%\FilmSheet\`
  - Linux: `~/.config/FilmSheet/`
